# Rota Inteligente: Otimização de Entregas com Algoritmos de IA para Sabor Express
##Projeto desenvolvido para a disciplina **Artificial Intelligence Fundamentals**.

## Estrutura do Projeto
- `/data`: dados de entrada (nós, arestas e pedidos)
- `/src`: código-fonte dos algoritmos (A*, K-Means, pipeline)
- `/docs`: imagens e resultados gerados

## Objetivo
Otimizar as rotas de entrega da empresa fictícia *Sabor Express* utilizando algoritmos de IA clássica (A* e K-Means) para sugerir rotas mais eficientes e agrupar entregas próximas.

## Como executar (futuramente)
```bash
pip install -r requirements.txt
python src/pipeline.py
```
